rdfs_from_owl() {|class_hash.size| ...}

This checks for owl:Classes and adds and rdfs:Class triple to them. This doesn‘t check if the triple already exists, and thus may cause duplicates. You can pass a block that will be called with the overall size as a parameter.

It returns the overall size, the number of modified elements and the number of blank nodes
